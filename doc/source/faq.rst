Hot keys
________

**General**

============  ==========
Hotkey        Action
============  ==========
Ctrl-Z        Undo
Ctrl-Y        Redo
Ctrl-X        Cut
Ctrl-C        Copy
Ctrl-V        Paste
Ctrl-S        Save
Ctrl-Shift-S  Save as 
Ctrl-O        Open
Ctrl-E        Export
============  ==========

**Map mode**

===============  =======================
Hotkey           Action
===============  =======================
1                Increase mode
2                Smooth mode
3                Level mode
4                Rectangle (crisp) increase mode
Mouse1           Use tool (inc. terrain)
Shift+Mouse1     Decrease terrain 
Ctrl+Mouse up    Increase size
Ctrl+Mouse down  Decrease size
Esc              Deselect tool
Mouse2           Deselect tool
===============  =======================
